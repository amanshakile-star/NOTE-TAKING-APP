# Quick Start Guide

## Browser Interface (Recommended)

1. Open terminal in `gemini-test` folder
2. Run: `npm run dev`
3. Open Chrome and go to: `http://localhost:3000`
4. Type your question and click "Send to Gemini AI"

## Terminal Mode

1. Open terminal in `gemini-test` folder
2. Run: `npm start`
3. See AI response in terminal

## List Available Models

Run: `npm run list-models`
